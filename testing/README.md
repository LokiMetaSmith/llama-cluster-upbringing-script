# Testing

This directory contains unit and integration tests for the various components of the project.

## Directory Structure

- **integration_tests**: Contains integration tests.
- **unit_tests**: Contains unit tests.
- **run_unit_tests.sh**: A script for running the unit tests.
- **test_distributed_llama.sh**: A script for testing the distributed llama.
- **test_duplicate_role_execution.sh**: A script for testing duplicate role execution.
- **test_paddler.sh**: A script for testing the paddler.
- **test_piper.sh**: A script for testing piper.
- **test_run.sh**: A script for running a test.
