# Debian Service

This directory contains files for setting up a Debian service.

## Directory Structure

- **default**: Default configuration files for the service.
- **init.d**: Init scripts for the service.
- **install_service.sh**: A script for installing the service.
- **MANUAL_TESTING_GUIDE.md**: A guide for manual testing of the service.
- **SERVICE_SETUP_AND_USAGE.md**: A guide for setting up and using the service.
