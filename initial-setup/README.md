# Initial Setup

This directory contains scripts for initial server setup.

## Directory Structure

- **modules**: Contains scripts that are called by `setup.sh`.
- **worker-setup**: Contains scripts for setting up a worker node.
- **add_new_worker.sh**: A script for adding a new worker.
- **setup.conf**: The configuration file for `setup.sh`.
- **setup.sh**: The main setup script.
- **update_inventory.sh**: A script for updating the inventory.
