# Initial Setup Modules

This directory contains scripts that are called by `setup.sh`.

## Files

- **01-network.sh**: A script for configuring the network.
- **02-hostname.sh**: A script for configuring the hostname.
- **03-user.sh**: A script for configuring the user.
- **04-ssh.sh**: A script for configuring SSH.
- **05-auto-provision.sh**: A script for auto-provisioning.
