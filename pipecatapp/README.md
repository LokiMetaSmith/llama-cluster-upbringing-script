# Docker Pipecat App

This directory contains the Dockerfile and related files for the pipecatapp application.

## Directory Structure

- **prompts**: Contains prompts for the agent and various experts.
- **static**: Contains static files for the web server.
- **tools**: Contains tools that can be used by the agent.
- **Dockerfile**: The Dockerfile for the application.
- **api_keys.py**: A file for storing API keys.
- **app.py**: The main application file.
- **expert_tracker.py**: A file for tracking experts.
- **llm_clients.py**: A file for LLM clients.
- **memory.py**: A file for managing the agent's memory.
- **moondream_detector.py**: A file for the moondream detector.
- **requirements.txt**: The Python requirements for the application.
- **start_pipecat.sh**: A script for starting the pipecat application.
- **test_server.py**: A file for testing the server.
- **web_server.py**: The web server for the application.
