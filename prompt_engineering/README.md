# Prompt Engineering

This directory contains scripts and tools for evaluating and improving the AI's prompts.

## Directory Structure

- **agents**: Contains agent definitions.
- **evaluation_suite**: Contains the evaluation suite for the prompts.
- **generated_evaluators**: Contains generated evaluators.
- **create_evaluator.py**: A script for creating a new evaluator.
- **evaluation_lib.py**: A library of functions for evaluation.
- **evaluator.py**: The main evaluator script.
- **evolve.py**: A script for evolving the prompts.
- **PROMPT_ENGINEERING.md**: A guide to prompt engineering.
- **requirements-dev.txt**: The development requirements for prompt engineering.
