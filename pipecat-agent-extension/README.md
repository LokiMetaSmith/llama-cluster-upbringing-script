# Pipecat Agent Extension

This directory contains a Gemini CLI extension for interacting with the agent.

## Directory Structure

- **commands**: Contains the commands for the extension.
- **dist**: Contains the compiled extension.
- **example.ts**: An example of how to use the extension.
- **gemini-extension.json**: The configuration file for the extension.
- **package.json**: The package configuration file.
- **tsconfig.json**: The TypeScript configuration file.
