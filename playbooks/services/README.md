# Ansible Service Playbooks

This directory contains playbooks for specific services.

## Files

- **ai_experts.yaml**: A playbook for AI experts.
- **app_services.yaml**: A playbook for application services.
- **consul.yaml**: A playbook for Consul.
- **core_ai_services.yaml**: A playbook for core AI services.
- **core_infra.yaml**: A playbook for core infrastructure.
- **docker.yaml**: A playbook for Docker.
- **final_verification.yaml**: A playbook for final verification.
- **model_services.yaml**: A playbook for model services.
- **nomad.yaml**: A playbook for Nomad.
